
=Email=

'''urn:org:netkernel:email:core''' provides a comprehensive set of
transports and services for sending and receiving email messages.

==Receiving==

Email messages may be received either directly to a local instance
of an SMTP server running within NetKernel or on a remote email
server.

*[[doc:email:core:SMTPTransport|SMTPTransport]]
*[[doc:email:core:MailFolderTransport|MailFolderTransport]]

==Sending==

Email messages can be sent via an SMTP gateway

*[[doc:email:core:sendmail|active:sendmail]]

==Acknowledgements==

The library is principally based on the javax.mail API.

The direct SMTPTransport server is based on the [http://code.google.com/p/subethasmtp|SubethaSMTP] project which is [[doc:email:core:subethasmtp:license|licensed]] under the Apache
